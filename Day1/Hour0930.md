# AI 기술 발전 흐름

## RNN (Recurrent Neural Network)

순환 신경망

특징

- 이전 입력 정보를 기억
- 시계열 데이터 처리
- 자연어 처리
- 음성 인식

한계

- 긴 문장 기억 어려움
- 병렬 처리 어려움

---

## Transformer

# Attention Is All You Need

AI 역사상 가장 영향력 있는 논문 중 하나

주요 주장

> RNN 없이 Attention만으로 자연어 처리가 가능하다.

## 기존 방식

RNN → LSTM → GRU

문장을 순차적으로 처리

## Transformer 방식

Attention 기반

문장 전체를 한 번에 분석

장점

- 병렬 처리 가능
- GPU 활용 극대화
- 긴 문맥 처리 가능
- 대규모 학습 가능

---

# Transformer 구조

## Encoder

입력 문장의 의미를 추출

구성

- Input Embedding
- Positional Encoding
- Multi-Head Attention
- Feed Forward

## Decoder

출력 문장을 생성

구성

- Masked Multi-Head Attention
- Encoder-Decoder Attention
- Feed Forward
- Softmax

---

# 핵심 개념

## Embedding

단어를 숫자 벡터로 변환

## Positional Encoding

단어 순서 정보 제공

## Attention

현재 단어와 관련된 단어에 집중

## Multi-Head Attention

여러 관점에서 Attention 수행

예

- 문법
- 의미
- 거리
- 문맥

## Masked Attention

미래 단어를 보지 못하도록 제한

# GPT와 Transformer의 관계

원본 Transformer

Encoder + Decoder

BERT

Encoder Only

GPT

Decoder Only

ChatGPT는 Transformer의 Decoder 부분을 확장한 구조
