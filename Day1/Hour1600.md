## System Prompt

```python
SYSTEM_PROMPT = """
너는 미국의 고전 코미디와 애보트와 코스텔로의
'Who's on First?' 개그를 매우 잘 알고 있는 코미디언이다.
사용자의 질문을 그 개그와 연결해서 답변하라.
"""
```

---

## 핵심 함수

```python
def get_response(question: str) -> str:
    response = client.chat.completions.create(
        model="dev-gpt-5.4-mini",
        messages=[
            {"role": "system", "content": SYSTEM_PROMPT},
            {"role": "user", "content": question},
        ]
    )
    return response.choices[0].message.content
```

---

## 반복문

```python
while True:
    question = input("궁금한 걸 물어보세요: (끝내려면 q 입력): ").strip()
    if question == 'q':
        break
    print(get_response(question))
```

---

## 핵심

```text
사용자 입력
↓

↓
Azure OpenAI 호출
↓
응답 텍스트 출력
↓
q 입력 시 종료
```

## 입력 처리

`poem.py`는 하나의 문자열을

```text
주제,내용
```

형태로 받아서 쉼표 기준으로 분리한다.

```python
texts = text.split(',')
if len(texts) >= 2:
    subject, content = texts[0].strip(), texts[1].strip()
else:
    subject, content = '', text
```

---

## Temperature

```python
temperature=0.9
```

시 생성은 정답형 작업이 아니라 창작형 작업이므로

조금 더 다양한 표현이 나오도록 temperature를 높게 설정

---

f"""
시의 주제는 '{subject}', 내용은 '{content}'야.

## 메시지 출력

```python
def append_message(sender, message):
    chat_log.config(state=tk.NORMAL)
    ...
    chat_log.config(state=tk.DISABLED)
    chat_log.see(tk.END)
```

읽기 전용 채팅창을 잠시 열어 메시지를 추가한 뒤

다시 비활성화한다.

---

## 전송 처리

```python
def send_message(event=None):
    user_input = input_box.get("1.0", tk.END).strip()
    if not user_input:
        return
```

입력값이 없으면 아무 작업도 하지 않는다.

---

## 챗봇과 시인 분기

```python
if "," in user_input:
    reply = poem.get_response(user_input)
    sender = "AI 시인"
else:
    reply = chatbot.get_response(user_input)
    sender = "ChatGPT"
```

입력에 쉼표가 있으면

```text
주제,내용
```

형태로 판단하여 AI 시인을 호출한다.

쉼표가 없으면 일반 챗봇으로 처리한다.

---

## Thread 사용

```python
threading.Thread(target=call_api, daemon=True).start()
```

API 호출 중 GUI가 멈추지 않도록

별도 스레드에서 Azure OpenAI를 호출한다.

---

## Tkinter 업데이트

```python
win.after(0, lambda: append_message(sender, reply))
win.after(0, lambda: send_btn.config(state=tk.NORMAL))
```

백그라운드 스레드에서 직접 UI를 수정하지 않고

Tkinter 메인 루프에 작업을 예약한다.
