## System Instructions vs User Prompt

### System Instructions

강사 표현

```text
헌법
```

---

특징

- 시스템이 정의
- 사용자보다 우선
- AI의 정체성 정의

---

### User Prompt

특징

- 사용자가 입력
- 작업 지시
- 일시적

---

### 핵심

```text
System = WHO

User = WHAT
```

## 판교걸즈 실습

질문

```text

```

---

초기 응답

```text
정보 없음
```

---

이후 정보 제공

```text

"판교에는 사과나무를 심자"
```

---

### 핵심

현재 대화에서 제공된 정보가

임시 지식처럼 활용될 수 있다.

```text

```
