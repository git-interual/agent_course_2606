# 1. LLM은 Stateless

LLM은 기본적으로 상태를 저장하지 않는다.

```text
질문
↓
응답
```

이 끝나면 이전 대화를 기억하지 못한다.

---

예시

사용자

```text
나는 김기용이야
```

↓

사용자

```text
내 이름이 뭐지?
```

---

LLM 단독으로는 답변할 수 없다.

---

## ChatGPT가 기억하는 이유

실제로는

```text
대화 이력 전체
```

를 함께 다시 전달하기 때문이다.

---

구조

```text
사용자
↓
대화 기록
↓
LLM
```

---

## 핵심

```text
LLM은 기억하지 않는다.

Agent가 기억한다.
```

---

# 6. Tool 함수 작성

예제

```python
from pydantic import Field
from typing import Annotated

def get_weather(
    location: Annotated[
        str,
        Field(description="도시 이름. 서울, 도쿄, 파리 등")
    ]
) -> str:

    conditions = ["맑음", "흐림", "비", "눈"]

    print(
        f"Tool get_weather called with location: {location}"
    )

    return (
        f"{location}의 오늘 날씨는 "
        f"{conditions[0]}입니다."
    )
```

---

## Annotated

Tool 파라미터 설명 제공

```python
location: Annotated[
    str,
    Field(description="도시 이름")
]
```

---

### 오류

```text
NameError:
Annotated is not defined
```

---

### 해결

```python
from typing import Annotated
from pydantic import Field
```

# 10. 현재 hello_agent.py

```python
from dotenv import load_dotenv
import os
import asyncio

from agent_framework.openai import OpenAIChatClient
from openai import AsyncAzureOpenAI

from pydantic import Field
from typing import Annotated

load_dotenv()


def get_weather(
    location: Annotated[
        str,
        Field(
            description=
            "도시 이름. 서울, 도쿄, 파리 등"
        )
    ]
) -> str:

    conditions = [
        "맑음",
        "흐림",
        "비",
        "눈"
    ]

    print(
        f"Tool get_weather called with location: "
        f"{location}"
    )

    return (
        f"{location}의 오늘 날씨는 "
        f"{conditions[0]}입니다."
    )


async def main() -> None:

    endpoint = os.getenv(
        "AZURE_OPENAI_ENDPOINT"
    )

    api_key = os.getenv(
        "AZURE_OPENAI_API_KEY"
    )

    model = os.getenv(
        "AZURE_OPENAI_CHAT_MODEL"
    )

    api_version = os.getenv(
        "AZURE_OPENAI_API_VERSION"
    )

    azure_client = AsyncAzureOpenAI(
        azure_endpoint=endpoint,
        api_key=api_key,
        api_version=api_version,
    )

    agent = OpenAIChatClient(
        async_client=azure_client,
        model=model,
    ).as_agent(
        name="HelloAgent",
        instructions=
            "너는 친절한 AI 비서야.",
        tools=[
            get_weather
        ],
    )

    result = await agent.run(
        "프랑스 파리 날씨는 어때?"
    )

    print(result)


if __name__ == "__main__":
    asyncio.run(main())
```

# 2교시 핵심 요약

```text
LLM은 Stateless

↓

Session 필요

↓

Memory 필요

↓

Context Window 한계

↓

Tool Calling

↓

Agent가 Tool 선택

↓

Tool 결과 기반 응답

↓

Tool 품질이 중요
```

---

## 오늘의 핵심 메시지

Agent는 단순히 답변을 생성하는 프로그램이 아니다.

Agent는

- 기억(Session)
- 메모리(Memory)
- 도구(Tools)

를 활용하여 문제를 해결하는 시스템이다.

특히 Tool Calling 능력은 현대 Agent의 핵심 경쟁력이다.
