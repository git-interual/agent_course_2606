# 3교시 핵심 메시지

Agent는 단순한 Chatbot이 아니다.

Agent는

```text
LLM

+

Session

+

Memory

+

Tools
```

를 조합하여

복합적인 작업을 수행하는 시스템이다.

특히 Session을 이용하면 Multi-turn 대화가 가능하며,

여러 Tool을 조합하여 사용자의 요구사항을 해결할 수 있다.
