# 3. Tool Decorator

새롭게 추가된 내용

```python
from agent_framework import tool
```

---

날씨 Tool

```python
@tool(approval_mode="never_require")
def get_weather(...):
```

---

환율 Tool

```python
@tool(approval_mode="never_require")
def get_exchange_money(...):
```

---

## 의미

이 함수는 Agent가 호출 가능한

공식 Tool임을 선언한다.

---

### approval_mode

```python
approval_mode="never_require"
```

---

의미

```text
Tool 호출 시

사용자 승인 없이

즉시 실행
```

---

# 4. Weather Tool 개선

기존

```python
return f"{location}의 오늘 날씨는 {conditions[index]}입니다."
```

---

개선

```python
ordinal_sum = sum(ord(c) for c in location)

temperature = (ordinal_sum % 35) + 5

return (
    f"{location}의 오늘 날씨는 "
    f"{conditions[index]}입니다. "
    f"기온은 {temperature}°C입니다."
)
```

---

## 특징

도시명마다

```text
날씨

+

기온
```

이 고정적으로 계산된다.

---

예시

```text
런던 → 천둥번개, 10°C

파리 → 구름 많음, 30°C
```

# 7. Console Chatbot 구현

기존

```python
await agent.run(...)
```

1회 실행

---

변경

```python
while True:
```

사용

---

코드

```python
while True:

    user_input = input("사용자: ")

    if user_input.lower() in [
        "exit",
        "quit"
    ]:
        break

    result = await agent.run(
        user_input,
        session=session
    )

    print(result)
```
