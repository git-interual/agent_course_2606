# 1. Data Types

AI Agent는 다양한 데이터 형식을 사용한다.

---

## Markdown

확장자

```text
.md
```

특징

- 사람이 읽기 쉬움
- LLM 친화적
- GitHub 기본 문서 형식

---

## JSON

JavaScript Object Notation

특징

- 구조화 데이터
- API 표준 형식
- Agent 입출력에 자주 사용

예시

```json
{
  "name": "Kim",
  "age": 20
}
```

---

## CSV

Comma Separated Values

특징

- 표 형태 데이터
- Excel 호환
- 데이터 분석에 활용

---

## XML

eXtensible Markup Language

특징

- 태그 기반 구조
- 기업 시스템에서 많이 사용
- 최근에는 JSON에 밀리는 추세

---

## 강의 메시지

Agent 개발에서는

```text
Markdown
JSON
CSV
```

를 가장 자주 다룬다.
