강사 소개

강의의 진행 방식과 실습 환경을 확인했습니다.

Transformer

RNN에서 Attention으로 넘어온 생성형 AI의 구조적 전환.

Token

문장이 토큰 시퀀스로 바뀌고 다음 토큰 예측으로 이어지는 과정.

Prompt Engineering

Role, Context, Task, Format을 조합해 모델의 출력을 설계.

개발환경의 이해

AI 개발과 운영에서 클라우드, 리소스, 배포가 갖는 의미.

Support Models

SLM, MoE, 경량화 기법으로 비용과 속도를 함께 고려.

Microsoft Foundry

Azure AI Foundry에서 모델을 탐색하고 배포하는 실습.

OpenAI 연결 정보

Azure OpenAI 연결 정보를 환경변수로 분리하고 Python SDK에서 사용.

반복문 챗봇 소스

사용자 입력을 반복 처리하는 콘솔 챗봇과 GUI 챗봇 구현.

인공지능 시인

주제와 내용을 받아 temperature를 높인 창작형 시 생성 실습.

Agent Framework

Microsoft Agent Framework의 구조와 LangChain, LangGraph와의 차이.

Memory & Tools

AgentSession, Multi-turn, Tool Calling으로 Agent 동작을 확장.

MVP Tour Agent

여행 상담 Persona, 날씨/환율 Tool, Session 기반 상담 흐름 구성.

RAG & Vector DB

Embedding, Retrieval, ChromaDB로 문서 기반 지식 검색 구현.

Real Function Calling

Mock Tool을 실제 날씨 API와 환율 API 호출로 확장.

Azure VM 운영

SLA, 가용성, SSH 접속, OpenClaw와 외부 채널 연동 흐름 확인.
