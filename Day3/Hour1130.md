# 2. 배포 설정 파일 생성

## .deployment

생성 파일

```ini
[config]
SCM_DO_BUILD_DURING_DEPLOYMENT=false
```

### 의미

Azure App Service 배포 시 자동 빌드를 수행하지 않는다.

```text
Azure 배포
↓
자동 빌드 생략
↓
배포된 파일 그대로 사용
```

---

## start.sh

생성 파일

```bash
pip install fastapi uvicorn
python -m uvicorn company_portal:app --reload --host 0.0.0.0 --port 8000
```

### 역할

FastAPI 서버 실행

```text
start.sh
↓
pip install
↓
uvicorn
↓
company_portal:app
```

---

## 참고

실무 운영 환경에서는

```bash
--reload
```

옵션을 제거하는 것이 일반적이다.

```bash
python -m uvicorn company_portal:app --host 0.0.0.0 --port 8000
```
